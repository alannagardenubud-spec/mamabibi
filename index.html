Welcome

This guide is designed to help you perform tasks common to license management. The primary audience for this guide is any site administrator responsible for installing and managing ArcGIS for Desktop, ArcGIS Engine, Esri CityEngine, ArcInfo Workstation, or ArcGIS Pro concurrent use, and ArcGIS Pro named user via an on-premise Portal for ArcGIS. This documentation will assist you in using ArcGIS License Manager and troubleshooting issues that may arise during its use. In the list of contents, click the topic you want to learn more about. You can also search for a particular topic using the Search window.

How to use this document

This document is divided into the sections below:

Getting started

The Getting started section provides installation and startup information that will help you get ArcGIS License Manager running. It also describes the process to set up the client machines to use the license manager.

Administering the License Manager

This section provides information that will assist you in using ArcGIS License Manager and managing your licenses. This section provides information on configuring license borrowing and viewing license availability.

ArcGIS License Manager for Portal for ArcGIS

In this section, instructions can be found for authorizing and configuring ArcGIS License Manager for use with named user ArcGIS Pro via an on-premise Portal for ArcGIS instance. Useful Information and helpful hints can be found in this topic.

Advanced topics

Refer to this section for complex license management issues, such as using the options file or failover license servers.

Troubleshooting

This section lists common problems with the license manager, probable causes, and suggested solutions. For example, if you have trouble connecting to a Windows license server, see Cannot connect to license server.

Glossary

The glossary contains definitions of commonly used ArcGIS copy protection terms.

Contacting Esri Customer Service

If for any reason you are unable to locate your authorization information, you can contact Esri Customer Service and request that it be sent again.

In the United States, you can contact Esri Customer Service by calling 888-377-4575 or visiting the My Esri Site.

Outside the United States, contact your local Esri distributor. For information regarding your local distributor, visit the following website.

Technical Support Knowledge Base
